# Product Overview

ARI 2.0 is a CRM and billing management system designed for government entities and municipalities to manage citizens/taxpayers, billing, payments, and workflows.

## Core Purpose
Manage customer/contributor information including personal data, legal representatives, contact information, addresses, identity documents, and relationships between actors in the system.

## Key Modules
- **CRM (Customer/Contributor Management)**: Personal information, phones, addresses, emails, identity cards, legal representatives, relationships
- **Billing/Payments**: Account management, obligations, payment structures (future)
- **Workflow**: Business process management (future)

## Architecture Vision
- Cloud and on-premise deployment capability
- AI-first design optimized for AI agent development
- Data Lake integration for analytics
- Microservices architecture (authentication, auditing, queuing)

## Current Status
Phase 1: CRM module with 21 entities fully implemented using Repository Pattern and Service Layer architecture.
